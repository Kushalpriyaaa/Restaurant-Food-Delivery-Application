### 2. Microservices-Inspired Backend

The backend is organized into modular services:
- **Auth Service** - User authentication and authorization
- **Menu Service** - Menu item management
- **Order Service** - Order processing and tracking
- **Delivery Service** - Delivery assignment and tracking
- **Offers Service** - Discount and promotion management
- **Reports Service** - Analytics and reporting
- **Settings Service** - Configuration management

### Data Security

1. **Row-Level Security**
   - Users can only access their own orders
   - Admins can access all data
   - Delivery personnel can only see assigned orders

2. **API Security**
   - All mutations require authentication
   - Role-based function access
   - Input validation on all endpoints

3. **Client-Side Protection**
   - ProtectedRoute components
   - Conditional rendering based on role
   - Secure token storage

1. **Client-Side**

   - Database indexing
   - Query optimization
   - Real-time subscriptions (no polling)
   - Convex's automatic optimization

   - CDN for static assets
   - Image compression
   - Gzip compression
   - HTTP/2 support
